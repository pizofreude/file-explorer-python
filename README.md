# file-explorer-python

Barebone windows file explorer written in Python.

## Main Functions

This project develop basic ways to:

1. Save file(s)
2. Open file(s)
3. Rename file(s)
4. Delete file(s)
5. Move file(s)
6. Create folder(s)
7. Deleting the folder(s)

## Libraries

1. tkinter - `pip install tkinter`
2. easygui - `pip install easygui`
